# 🌌 Horizon

A modern, animation-rich web experience inspired by Zentry.  
Built to explore advanced UI/UX interactions using GSAP and modern frontend tools.

---

## 🚀 About the Project

**Horizon** is a visually immersive website focused on smooth animations, transitions, and interactive storytelling.  
The project was primarily created to deepen understanding of **GSAP** and high-end frontend animation techniques.

This is currently a **frontend-only project** — backend functionality is not implemented yet.

---

## ✨ Features

- 🎬 Smooth scroll-based animations
- ⚡ High-performance transitions using GSAP
- 🎨 Modern UI inspired by premium web experiences
- 📱 Responsive design
- 🧩 Component-based architecture with Next.js
- 🌈 Styled using Tailwind CSS

---

## 🛠️ Tech Stack

- **Framework:** Next.js  
- **Animations:** GSAP (GreenSock Animation Platform)  
- **Styling:** Tailwind CSS  
- **Language:** TypeScript

---

## 🎯 Purpose

This project was built to:

- Master **GSAP animations**
- Understand **timeline-based animation flows**
- Practice **modern UI/UX design patterns**
- Explore **interactive storytelling on the web**
